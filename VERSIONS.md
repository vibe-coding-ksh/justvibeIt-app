- Windows: winget auto-install for Git, Node.js, GitHub CLI (no more manual download pages)
- macOS: DMG universal build (Intel + Apple Silicon)
- Windows: NSIS installer + portable EXE build config
- Cross-platform window fixes (titleBarStyle, npm.cmd, Git Bash fallback)
- README bilingual support (English + Korean)
